# User Guide

## How to Use the Assistant

1. Start the app with:

```bash
streamlit run app.py
```
2. Choose a learning mode.
3. Enter a process engineering question.
4. Review the answer.
5. Ask follow-up questions or request examples.
6. Learning Modes
7. Tutor

Best for concept explanations.

Quiz Generator

Creates practice questions.

Worked Example

Generates step-by-step engineering examples.

Interview Preparation

Explains concepts in a professional interview style.

Industrial Application

Connects theory to real industrial use cases.

Roadmap Builder

Creates structured learning plans
